API Documentation

1. Get a list of cities

/cities

Request method: GET
Returns a list of city objects

http://easy-fly.herokuapp.com/cities/

2. Sign up

/signup

Request method: POST
Required URL parameters: name, email, password
Returns a JsonResponse with containing the user_token

http://easy-fly.herokuapp.com/signup?name="bob"&email="bobhitler@gmail.com&password="ihatepasswords123"

3. Log in

/login/

Request method: POST
Required URL parameters: email, password
Returns a JsonResponse with containing the user_token

http://easy-fly.herokuapp.com/login/?email="bobhitler@gmail.com&password="ihatepasswords123"

4. Get All Available Flights

/login/

Request method: GET
Required URL parameter: auth_token (for authentication)

http://easy-fly.herokuapp.com/flights/?auth_token=user_token

0. Book a flight

/book-flight/

Request method: GET
Required URL parameter: flight_id, payment_status, auth_token(for authentication)
Returns user's ticket code for the booked flight.

http://easy-fly.herokuapp.com/book-flight/?auth_token=user_token&flight_id=23&payment_status="successful"
